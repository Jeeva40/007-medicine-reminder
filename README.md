💊 Medicine Reminder

A modern and responsive Medicine Reminder web application built with HTML, CSS, and JavaScript. It helps users manage medications, track adherence, and receive timely reminders through a simple and intuitive interface.

✨ Features
Add and manage medicines
Mark medicines as taken or skipped
Countdown to the next reminder
Track medication progress
Browser notifications
Local Storage support
Responsive design
Dark mode
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
🚀 Getting Started
Clone the repository.
Open the project in Visual Studio Code.
Launch index.html in your browser or use Live Server.
